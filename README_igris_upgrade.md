## 📝 About This Repository

**ASCENT OF THE SHADOW MONARCH** is a structured, gamified learning repository documenting my journey from core Python programming to Machine Learning engineering readiness. Inspired by the Solo Leveling progression system, every completed task unlocks new pixel art — evolving the character's outfit and rank as skills grow.

### 🎯 What You'll Find Here

- **🐍 Python Foundations** — OOP, functional programming, decorators, generators, file handling, type hinting
- **📊 Data Science Stack** — NumPy for numerical computing, Pandas for data manipulation, Matplotlib & Seaborn for visualization
- **🤖 Machine Learning** — Supervised and unsupervised algorithms with Scikit-Learn, preprocessing, model evaluation, pipelines
- **💡 Interview Preparation** — Curated ML/AI engineering interview questions across theory, coding, system design, and mathematics
- **🗡️ Shadow Monarch Evolution** — A 7-stage pixel art character that evolves as you complete each phase

> 💡 **Tip:** Every `git push` that completes a task unlocks new pixels and evolves the Shadow Monarch's outfit. Complete all stages to earn the final promotion to Advanced ML!

```python
        result = func(*args, **kwargs)
        print(f"{func.__name__} took {time.time() - start:.4f}s")
        return result
    return wrapper
```

```yaml
name: Shadow Monarch Evolution
on:
  push:

jobs:
  evolve:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.10'

        run: python scripts/update_pixel_art.py
      - name: Commit changes
        run: |
          git config --global user.name "Shadow Monarch"
          git config --global user.email "monarch@shadow.level"
```
